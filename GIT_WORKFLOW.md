# Git Workflow - Quick Reference

## Pushing Updates to GitHub

Whenever you make changes and want to push them to GitHub, run these 3 commands:

```powershell
git add .
git commit -m "Describe what you changed"
git push origin main
```

## Example

```powershell
# After editing calculate_citation_rates.py:

git add .
git commit -m "Fix citation calculation bug"
git push origin main
```
